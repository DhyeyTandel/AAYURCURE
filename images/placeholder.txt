## Image Guidelines:
- Use JPG for photos, PNG for graphics with transparency
- Optimize images for web (compress to reduce file size)
- Recommended sizes:
  - Hero images: 1920x1080px
  - Profile photos: 400x400px
  - Treatment images: 800x600px
  - Thumbnails: 300x200px
